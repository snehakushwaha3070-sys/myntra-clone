# Myntra Clone 🛍️

A front-end clone of the popular Indian e-commerce fashion website [Myntra](https://www.myntra.com/), built using pure HTML and CSS.

## 🛠️ Tech Stack
- HTML5
- CSS3
- Google Material Symbols (icons)
- Google Fonts

## ✨ Features
- Myntra-style header with logo
- Navigation bar (Men, Women, Kids, Home & Living, Beauty, Studio)
- Search bar with icon
- Action icons — Profile, Wishlist, Bag
- Banner section
- Footer
